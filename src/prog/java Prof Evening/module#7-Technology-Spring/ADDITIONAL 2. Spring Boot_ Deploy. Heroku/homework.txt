1. Создайте клиента с полями id, name, email и phone.
2.В Service создайте 5 методов:
      1.Создание нового клиента
      2.Возвращение списков всех клиентов
      3.Возвращение клиентов по его ID
      4.Обновление клиентов с заданным ID
      5.Удаление клиента с заданным ID

3. Подумайте над созданием контроллера и выполните все требования
4. Проверьте свои запросы с помощью браузера или же с помощью Postman

Необходимо разработать сервис "Аптека", в котором есть 2 роли:
1. Клиент
2. Продавец

В аптеке также есть товары. У товаров есть:
1.Название
2.Цена
3.Без рецепта/ С рецептом

Клиент после успешной авторизации может:

1.Посмотреть список всех товаров
2.Получить информацию о товаре по названию
3.Получить информацию о товаре по цене

Продавец после успешной авторизации может:

1.Добавить товар
2.Обновить товар
3.Удалить товар
4.Посмотреть список товаров

Реализуйте данный сервис с использованием аутентификации JWT.

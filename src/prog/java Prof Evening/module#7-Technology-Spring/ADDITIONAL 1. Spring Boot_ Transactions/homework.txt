Во всех заданиях используйте транзакции.

1. Напишите метод, который снимает деньги с банковского счета и обновляет баланс счета.

2. Реализуйте метод, который переводит деньги с одного банковского счета на другой.
Обновление баланса обоих счетов должно происходить одновременно (паралелльно).

3. Напишите метод, который сохраняет в базе данных новую запись о клиенте и новую запись о банковском счете.
Обе записи должны сохраняться вместе. Транзакция будет откатана, если произойдет ошибка при сохранении одной из записей.

4. Создайте метод, который обновляет баланс банковского счета на основе введенного пользователем значения.
Если новый баланс будет отрицательным, выбросьте исключение, чтобы предотвратить обновление. 

5. Реализуйте метод, который обновляет баланс нескольких банковских счетов в одной транзакции.
Обновления должны происходить вместе и транзакция должна откатываться,
если какое-либо из обновлений не удалось.
